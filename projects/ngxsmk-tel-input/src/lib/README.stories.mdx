## Quick Start

```typescript
import { NgxsmkTelInputComponent } from 'ngxsmk-tel-input';
import { ReactiveFormsModule, FormBuilder } from '@angular/forms';

@Component({
  imports: [ReactiveFormsModule, NgxsmkTelInputComponent],
  template: `
    <ngxsmk-tel-input
      formControlName="phone"
      label="Phone Number"
      [initialCountry]="'US'"
      [preferredCountries]="['US', 'GB']"
    />
  `
})
export class MyComponent {
  form = this.fb.group({
    phone: ['', Validators.required]
  });
}
```

## Installation

```bash
npm install ngxsmk-tel-input intl-tel-input libphonenumber-js
```

Or use the Angular CLI schematic:

```bash
ng add ngxsmk-tel-input
```
